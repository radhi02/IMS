## ERP System

An ERP system created using PHP Laravel, AJAX, jQuery, MySQL, JavaScript, and ChartJS. With a user-friendly interface, it allows users to access real-time data, generate reports, and make informed decisions. The use of AJAX and jQuery enhances the system's performance and enables dynamic page updates, while the use of ChartJS facilitates the creation of interactive data visualizations. Overall, this ERP system provides a robust and scalable solution for managing all aspects of a business. It has multiple modules, such as:

- Lead Generation.
- Sales Order.
- Material Requirement.
- Purchase Order.
- Material Receive.
- Manufacturing Order.
- Invoice.
